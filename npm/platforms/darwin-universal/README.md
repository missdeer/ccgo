# @ace-tool-rs/darwin-universal

Platform-specific binary for [ace-tool-rs](https://github.com/missdeer/ace-tool-rs) (macOS Universal - x64 and arm64).

## Overview

This package contains the pre-built binary for macOS (Universal). It is automatically installed as an optional dependency when you run `npx ace-tool-rs` on macOS.

**Do not install this package directly.** Instead, use:

```bash
npx ace-tool-rs --base-url <API_URL> --token <AUTH_TOKEN>
```

## Supported Platforms

- macOS x64 (Intel)
- macOS arm64 (Apple Silicon)
