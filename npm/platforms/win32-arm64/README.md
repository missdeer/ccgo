# @ace-tool-rs/win32-arm64

Platform-specific binary for [ace-tool-rs](https://github.com/missdeer/ace-tool-rs) (Windows ARM64).

## Overview

This package contains the pre-built binary for Windows ARM64. It is automatically installed as an optional dependency when you run `npx ace-tool-rs` on Windows ARM64 systems.

**Do not install this package directly.** Instead, use:

```bash
npx ace-tool-rs --base-url <API_URL> --token <AUTH_TOKEN>
```

## Supported Platforms

- Windows arm64
